# 📌 What is SQL?

**SQL (Structured Query Language)** is the standard language used to interact with relational databases. It allows us to create, retrieve, update, and delete data efficiently.

SQL is widely used by developers, data analysts, data scientists, and database administrators to manage and analyze data.

---

# 📌 What is Data?

Data refers to any piece of information that can be stored and processed.

Examples of data include:
- Student records
- Employee details
- Bank transactions
- Images
- Videos
- Documents
- Text messages

Without proper organization, data becomes difficult to manage.

---

# 📌 What is a Database?

A **Database** is a structured collection of related data stored electronically.

It helps users:
- Store data safely
- Retrieve information quickly
- Update existing records
- Analyze large amounts of information

### Examples
- School Management System
- Banking System
- Hospital Management System
- E-commerce Websites

---

# 📌 What is DBMS?

A **Database Management System (DBMS)** is software that allows users to create, store, organize, and manage databases.

Some popular DBMS software:
- PostgreSQL
- MySQL
- Oracle Database
- Microsoft SQL Server

---

# 📌 What is RDBMS?

**RDBMS (Relational Database Management System)** stores data in the form of tables consisting of rows and columns.

Each table can be related to another table using keys, making data organized and reducing redundancy.

Popular RDBMS:
- PostgreSQL
- MySQL
- Oracle
- SQL Server

---

# 📌 Why Do We Use SQL?

SQL makes working with data much easier.

Benefits of SQL:

- Update records efficiently

- Perform data analysis
- Supported by almost every relational database

# 📌 Table Structure

A database stores information inside **tables**.

Each table contains:

| Rows | Columns |
|------|----------|
| Records | Attributes |

Example:

| Student_ID | Name | Age |
|------------|------|-----|
| 101 | Rahul | 20 |
| 102 | Priya | 21 |

---

# 📌 Is SQL the Same in Every Database?

Most SQL commands are standard across different database systems.

However, some databases provide additional features and syntax specific to their platform.

For example:
- PostgreSQL
- MySQL
- SQL Server
- Oracle

Learning standard SQL makes it easier to switch between different database systems.
